Three broadcast address types. Only directed broadcast is theoretically routable — and even that is disabled by default on all modern routers (RFC 2644, 1999).
TypeAddressScopeRouter forwards?
Limited broadcast 255.255.255.255 Current subnet only NEVER
Directed broadcast 192.168.1.255 Named subnet DISABLED (RFC 2644)
Network broadcast (legacy) 192.168.1.0 Subnet (old BSD) NEVER
Multicast (not broadcast) 224.0.0.0 – 239.255.255.255 Group-scoped YES — with IGMP
Smurf attack (1997): attacker sends ICMP to a directed broadcast address, spoofing the victim's IP as source. Every host on that subnet replies to the victim — instant DDoS amplification by 200–300×. RFC 2644 closed this by requiring routers to drop directed broadcasts by default.
